mpas_analysis.__main__.build_analysis_list¶

mpas_analysis.__main__.build_analysis_list(config, controlConfig)[source]¶

Build a list of analysis tasks. New tasks should be added here, following the approach used for existing analysis tasks.

Parameters
configMpasAnalysisConfigParser object

contains config options

controlConfigMpasAnalysisConfigParser object

contains config options for a control run, or None if no config file for a control run was specified

Returns
analyseslist of AnalysisTask objects

A list of all analysis tasks
